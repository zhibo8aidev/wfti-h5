# WFTI H5 Design

## 1. 设计总述
- 项目：世界杯球迷人格大鉴赏（WFTI）H5 活动
- 设计载体：H5 / mobile-web / App WebView
- 已锁定视觉风格：A 高阶 3D 风格建模
- 目标气质：高亮影棚、商业级成年人审美、世界杯戏剧张力、人格识别感、强传播属性

## 2. 全局视觉系统
### 2.1 视觉母题
将整套 H5 定义为“高亮影棚中的人格鉴赏仪式”。场景核心由球形聚光穹顶、镜面地板、半透明数据卡、悬浮人格徽章组成，形成带戏剧感的世界杯人格秀场，而不是传统功能页。

### 2.2 KV 3D 场景设定
- 场景关键词：世界杯决赛夜、灯光升维、玻璃奖杯感、金属球面、人格舞台、镜面地板
- 主体元素：悬浮足球切面核心、人格面具徽章、环形灯架、半透明数据丝带、球场草纹被抽象为影棚地板反射
- 光线方式：高亮棚拍主光 + 冷暖双侧补光 + 底部镜面返光
- 画面禁区：纯黑背景、儿童动画、卡通大眼、过饱和糖果配色、低清贴图

### 2.3 底色取样策略
- KV 宽度固定为 `390px`
- KV 底部真实采样色定义为 `--floor-rgb: 224, 231, 240`
- 全页统一底色变量：`--page-floor: rgb(var(--floor-rgb))`
- KV 容器、`body`、卡片缝隙、底部环境层、异常态背景均共用同一底色体系
- 通过底部 18% 的珍珠灰到银蓝灰渐变建立“压舱地板”，保证长屏没有 1px seam 与底部视觉真空

### 2.4 标题色彩融合
- 标题禁用绝对黑
- 标题主色使用 `#31455D`，来自 KV 底部冷灰蓝取样
- 标题下方叠加 12% 白色弥散 scrim，保证在复杂 3D 背景上仍有高级通透感
- 中文标题使用厚重几何黑体，辅以轻微 3D 挤压和高光描边

### 2.5 色板
- 空间主色：`#F4F7FB`
- 影棚地板：`#E0E7F0`
- 冷金属蓝：`#6A88B8`
- 高光冰蓝：`#CDE6FF`
- 冠军金：`#F3C96B`
- CTA 热能橙：`#FF8F61`
- 深文本蓝灰：`#31455D`
- 辅助说明：`#64748B`

### 2.6 材质语言
- 玻璃态卡片：18-22px 背景模糊，白色 60%-76% 透明面板
- 金属边框：1px 半透明描边 + 顶部内高光
- 阴影：大半径扩散阴影，避免厚重硬阴影
- 装饰粒子：最多 3 个，透明度 0.05-0.1，只做环境沉积

## 3. 页面设计说明
### 3.1 活动落地页
视觉目标：用高阶 3D KV 快速建立“世界杯人格鉴赏秀场”的高级感，在首屏内完成情绪拉满、玩法理解、CTA 转化。

主要组件：
- 顶部 Logo 锚点，采用亮背景适配的彩色版素材
- 390px 宽 3D KV，主体为悬浮足球人格核心与环形灯架
- 主标题 `世界杯球迷人格大鉴赏`
- 单行 Slogan `你不是在看世界杯，你是在演你自己。`
- 层叠信息卡：16 道题 / 3 分钟 / 16 型人格 / 已有参与人数
- 底部悬浮 CTA 栏：开始鉴赏 / 查看我的结果（已测用户）
- 社交背书：`已有 128,430 位吧友完成鉴赏`

布局要点：
- 信息卡以 `-88px` 负边距压入 KV 底部
- 核心交互场锚定在屏幕下 40% 区域
- 长屏下通过扩展环境地板、数据粒子和参与人数条避免底部真空

### 3.2 答题页
视觉目标：保持仪式感连续，同时把认知负担压到最低，让 16 题作答流畅、稳、可回退。

主要组件：
- 顶部返回按钮 + 品牌胶囊
- 进度条组件，显示百分比与 `N/16`
- 题目卡：题号、维度标签、题干、辅助描述
- 三个大尺寸选项卡，点击后高亮并自动切题
- 底部缓存提示：`24h 内可继续作答`

设计处理：
- 题卡使用厚玻璃卡，选项卡做悬浮式分层
- 进度条用冰蓝能量条，已答区带轻微流光
- 切题过渡使用左右滑移 + 轻缩放，不用夸张翻页

### 3.3 结果计算过渡页
视觉目标：把算法等待包装成“人格扫描仪式”，让 2.5 秒不显空、不显技术 loading。

主要组件：
- 中央 3D 人格核心球
- 外圈分析环、扫描线、维度标签流
- 状态文案：`正在分析你的球迷 DNA……`
- 次级提示：隐藏人格检测优先于常规人格匹配

设计处理：
- 中心球缓慢呼吸缩放
- 环形轨道持续旋转
- 下方 8 维 chips 交替亮起，暗示本地计算进行中

### 3.4 结果页
视觉目标：让人格结果具有“被鉴定出来”的权威感，同时让用户愿意停留、截图和分享。

主要组件：
- 人格编号 + 中文人格名 + 英文代号
- 人格插画区
- 标签云
- 一句话人格结论
- 八维雷达图卡
- 深度解析卡
- 球星对照 / 天敌 / 拍档三联卡
- 二维码与品牌信息区
- 固定底部操作栏：保存相册 / 分享名片 / 重新预测

人格插画区域与标签云处理方式：
- 插画区采用“半身 3D 人格化角色 + 悬浮足球战术道具”的组合，不做纯平头像
- 角色占比约为内容首屏的 42%，背景使用同人格色系环境辉光
- 标签云以 6-8 个胶囊标签环绕角色分布，分为核心标签和戏剧标签
- 标签采用不同层级尺寸，避免机械整齐排布，形成“被人格场吸附”的感觉
- 标签云不压脸，不遮挡人格名；优先环绕肩部、腰侧、雷达图上沿

### 3.5 分享海报态
视觉目标：生成可以直接传播的成品，而不是结果页截屏替代物。

分享海报构图与二维码区域：
- 海报比例建议 `1170 x 2080`
- 上部 42% 为人格 3D 主视觉区，放人格代号、昵称与一句话标签
- 中部 36% 为雷达图、人格解析、球星对照
- 下部 22% 为二维码区、品牌 Logo、传播引导文案
- 二维码固定右下角，置于独立磨砂圆角盒中，确保扫描成功率和前端切图稳定性
- 二维码盒左侧放 `扫码查看你的球迷人格`，右侧为 96px 视觉安全区

### 3.6 异常态
视觉目标：异常页也要在同一套影棚世界观内，不能退化成通用系统白页。

覆盖状态：
- 活动未开始
- 活动已结束
- 网络异常

主要组件：
- 简化版 3D 场景插画
- 状态标题与说明
- 主 CTA：重试 / 返回首页 / 订阅提醒
- 已结束页附带热门人格 TOP3 小卡片，承接内容空白

## 4. CTA 触感系统与底部悬浮栏规范
### 4.1 CTA 触感系统
- 主按钮采用 320° 微渐变，主色从 `#FFA56F` 到 `#FF8F61`
- 顶层添加 `inset 0 1px 0 rgba(255,255,255,0.42)` 高光
- 外层阴影为 `0 18px 40px rgba(255,143,97,0.26)`
- 点击态缩放至 `0.985`，并抬升高光亮度

### 4.2 底部悬浮栏规范
- 统一为毛玻璃悬浮底栏，距屏幕底边 12-16px
- 圆角 `28px`，内边距遵循单手点击热区
- 落地页为单 CTA 或双 CTA
- 结果页为三按钮结构，其中主按钮视觉权重最高
- 底栏背景色必须吸附全页地板色，避免像贴片悬空

## 5. 长屏设备防真空方案
- 首屏内容层高度控制在长屏 85% 覆盖率以上
- 页面底部建立压舱式环境渐变，颜色从 `#E8EEF5` 过渡到 `#CBD5E1`
- 参与人数、品牌说明、地板反射线条作为低密度填充
- 所有主页面都保留底部延展装饰，不允许出现连续超过 `30vh` 的无设计纯底色区域
- 关键 CTA 区、社交背书区、二维码区优先使用 `margin-top: auto` 锚定到屏底

## 6. 组件规格摘要
### 6.1 卡片
- 大卡圆角：`32px`
- 中卡圆角：`24px`
- 与 KV 重叠负边距：`-80px` 到 `-104px`

### 6.2 字体层级
- Hero 标题：34/40，粗体，字距 -0.5
- 人格名：28/34，粗体
- 模块标题：18/24，半粗
- 正文：15/24，字距 0.3-0.5
- 辅助说明：13/20，颜色不浅于 `#94A3B8`

### 6.3 动效节奏
- 页面入场：420ms 淡入上浮
- 题目切换：400ms
- CTA 按压：160ms
- 过渡页分析环：线性无限

## 7. 开发落地备注
- HTML mockup 中统一暴露底色变量，供后续前端实现对齐
- 结果页二维码与海报二维码均采用独立安全区盒模型
- 海报生成时需要隐藏底部操作栏，保留海报专属画布
- 所有异常态保留与主视觉一致的背景变量与影棚地板
